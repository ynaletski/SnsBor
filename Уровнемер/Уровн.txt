Для программирования адреса:

Задать тип протокола 2 для соответствующего СОМ порта
например для COM3:

CP 0 0 2

Подключить только один уровнемер.

Ввести команду
COMx (COM3 напр.)

Из DspHost послать в  binary mode file для адреса 1,2,3:

A0_WR_ADR1
A0_WR_ADR2
A0_WR_ADR3
